# 🏢 Business Blog Card - devChallenges

Este proyecto es un componente de tarjeta de blog diseñado para perfiles empresariales. El reto se centró en lograr una estructura limpia y profesional, aplicando técnicas de diseño responsivo y optimización de recursos CSS.

## 🚀 Descripción del Proyecto

La **Business Blog Card** es una interfaz que presenta información relevante sobre artículos o servicios. Destaca por su equilibrio visual, tipografía legible y una adaptación fluida a diferentes tamaños de pantalla, desde dispositivos móviles hasta monitores de escritorio.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica avanzada.
* **CSS3:** Estilos modernos, incluyendo Flexbox para alineación y posicionamiento.
* **Google Fonts:** Implementación de las familias tipográficas *Lato* y *Poppins*.

## ✨ Características Principales

* **Diseño Responsivo:** La tarjeta ajusta su ancho dinámicamente y se limita en pantallas grandes para mantener la estética.
* **Interfaz Limpia (Clean UI):** Uso de sombras suaves (`box-shadow`) y bordes redondeados para un aspecto moderno.
* **Efectos Visuales en Imágenes:** Implementación de bordes curvos personalizados en la parte inferior de la imagen principal para romper la rigidez del diseño cuadrado.
* **Arquitectura de Código:** Uso de selectores de clase para mantener una baja especificidad y facilitar el mantenimiento.

## 🧠 Conceptos Clave Aplicados

* **Flexbox Layout:** Utilizado para la distribución del contenido en la tarjeta y el centrado absoluto del componente en el `viewport`.
* **Herencia en CSS:** Optimización del código mediante la definición de propiedades globales en el selector `body`, evitando la redundancia de estilos.
* **Box Model:** Control preciso de rellenos (`padding`) y márgenes para garantizar que el contenido no se desborde.



## 📦 Instrucciones de Instalación y Uso

Para ejecutar este proyecto de forma local, sigue estos pasos:

1.  **Clona el repositorio**
2.  **Estructura de archivos:**
    Asegúrate de que las imágenes se encuentren en la carpeta `/img` para que las rutas funcionen correctamente:
    ```text
    ├── index.html
    ├── styles.css
    └── /img
        ├── hero-image-1.jpg
        ├── avatar-image-business-card.png
        └── favicon.ico
    ```
3.  **Ejecución:**
    Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Firefox, Edge o Safari).
